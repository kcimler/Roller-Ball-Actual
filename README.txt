*Roller Ball README*

Game Name: Roller Ball

Objective:  Contolling a ball, navigate yourself across the stage into the goal.
			If you fall off or run out of time, you lose a life and have to retry
			the stage. If you run out of lives, you lose the game and start back 
			from the beginning. 

			If you can get through all of the stages without losing all of your lives
			you win!

Controls:
	Ball movement: Arrow Keys or the "WASD" keys.
	Camera Movement: Mouse
